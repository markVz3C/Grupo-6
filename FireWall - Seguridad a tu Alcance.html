      Seguridad a tu Alcance

      Martin loncon, Erick maturana, lenneth naveas, damian oyarce, Miguel aguilar

      ¿Que es?
      Un firewall es una herramienta de seguridad de red que controla el tráfico entre diferentes redes o entre un dispositivo y una red. Su propósito principal es proteger los sistemas de redes privadas o dispositivos individuales de accesos no autorizados, ataques y otras amenazas externas.
      Explicacion visual
       
      ¿Por que es necesario?
      El firewall es necesario porque actúa como una barrera de seguridad entre tu red interna y las amenazas externas, como hackers o software malicioso. Filtra y controla el tráfico de red, permitiendo solo las conexiones seguras y bloqueando las peligrosas, protegiendo así tus datos y dispositivos de posibles ataques.

      • All

      Filtrado de Paquetes

      Analiza los paquetes de datos entrantes y salientes basndose en reglas predefinidas, como direcciones IP, puertos y protocolos. Permite o bloquea el trfico segn estas reglas.

      Control de Aplicaciones

      Permite o bloquea aplicaciones especficas y controla el uso de aplicaciones basadas en la web o en la nube. Esto ayuda a prevenir el acceso a aplicaciones no autorizadas o peligrosas.

      Filtrado de Contenidos

      Bloquea o permite el acceso a ciertos tipos de contenido web, como sitios de redes sociales, sitios de descargas o contenidos explcitos. Es til para implementar polticas de uso aceptable.
